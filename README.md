# مشروع Grades (نظام التقديرات المتكامل)

هذا المشروع هو عبارة عن برنامج بسيط وموثق، مصمم بناءً على مبادئ التفكير البرمجي، للتعرف على تقدير الطالب بناءً على درجته النهائية في الاختبار.

## محتويات المشروع (مثل الأستاذ):

*   `algorithm.txt`: خوارزمية الحل بشكل نصي.
*   `deepseek_mermaid.txt`: كود مخطط التدفق (Flowchart).
*   `grade_program.py`: الكود البرمجي بلغة Python.
*   `README.md`: هذا الملف (وصف المشروع).

## مميزات البرنامج:

*   **تعليمات للمستخدم:** تظهر عند البداية.
*   **الاستمرارية:** يعمل البرنامج بشكل متكرر حتى يكتب المستخدم "exit".
*   **معالجة الأخطاء:** يتغلب على المدخلات غير الرقمية (ValueError) والدرجات خارج النطاق (0-100).
*   **التوثيق:** الكود البرمجي يحتوي على تعليقات توضح طريقة عمله.
